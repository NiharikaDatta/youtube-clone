# youtube-clone
Built a youtube clone in five days as first project after self-learning html+css
## Live Demo
[Click here to view!](https://niharikadatta.github.io/youtube-clone/)
## What I learnt:
- HTML fundamentals
  - different tags (nav, header etc)
- CSS fundamentals
  - Flexbox

  - Opacity
  - Hover effects
  - Absolute vs fixed positions

- Responsiveness
## Features
- Tooltips
- Hover effects
- Accurate replica of my own YouTube home landing page
- Pure HTML+CSS : No frameworks
